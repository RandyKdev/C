Detailed Description

Printing the words contained in a file named file.txt in alphabetical order and also their frequencies in to another file "wordcount.txt".

Given a file (file.txt) containing words (like a publication or a novel), where words are separated by a space, newline, hyphen or underscore. This program prints (writes or outputs) to another file (wordcount.txt), the individual words contained in file.txt with their frequencies (number of occurrences) each on a newline and in alphabetical order. This program uses the binary tree data structure to accomplish this task.
